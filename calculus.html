1. Differentiation aka Derivatives

This isn't really calculus, I know, but this is how far I've gotten in math (precalculus). Also, because of limits to how I can input mathematical equations, I can only teach you differentiation. Let's move on to the actual math.

What differentiation is really about is finding out the gradient of a curve at any fixed point that lies on it. Differentiation is usually denoted as dy / dx. Let me provide an example of differentiation. Let's say we have the function y = x^2 + 2x + 1 plotted on a graph. If you don't know what the graph looks like, input the function into this free graphing site online: Desmos Graphing Calculator

The graph should look like a parabola shaoed like an extreme smiley face. Let's pick a point that lies exactly on the curve. How about (2,9)? It lies on the curve and is easy to calculate derivatives from. Remember this simple rule when differentiating: any function x^n has a derivative of nx^n - 1, and any function mx^n has a derivative of mnx^n - 1. dy / dx for the function we want to use would be 2x + 2. The "1" in the function disappears because it is basically x^0, which would become 0x^-1, which is nothing. To find the gradient, we must make a tangent of the same gradient (a straight line that passes through a point on a curve and has the same gradient as the point on the curve). The function of the tangent would be y = 2x + 2. This is in the format y = mx + c, where m is the gradient, so the gradient of the line at that point is 2, as easy as that!
