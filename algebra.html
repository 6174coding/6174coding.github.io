Algebra

The basics

1. Intro & Simultaneous Equations

You might think algebra is a freakish type of math that flings random x's and y's at you, and that people always back away from, but no. It's actually quite simple, and can be very useful in many math problems you are given in class. Here is a very simple algebraic problem that anyone would know how to solve: x + 22 = 45. In this case, x is a variable, which means it could be any number until you know its value. In order to solve this problem, you have to use this simple rule: whatever mathematical operation you use on one side of the equation, you must use it on the other side as well. Another good thing to do is to isolate x from other numbers. In the equation x + 22 = 45, x has 22 added to it, and we only want the x part of x + 22 on the left side of the equation, so what we can do is eliminate 22 from that side by subtracting 22. Now we only have x on the left side. But, remember the rule I mentioned earlier? Because we subtracted 22 from the left side, we need to do that on the right side too. Because the right side is just 45, we just do 45 - 22, and that equals 23. Finally, we get x = 23. We have found a value for x (23), which is the only value that works for the equation to be true! Now for some more fun: simultaneous equations. This usually involves two equations with two variables: x and y, in this case. Why they are called simultaneous equations is because you have to solve for x and y at almost the same time. There are two good methods for solving them: substitution, and addition/subtraction of entire equations. Let's use this pair of equations, though you may need to know this first: 2x is really x * 2, 3x is really x * 3, etc. Here we go...

  1. x + y = 14
  2. 4x - 5y = 2

To use substitution for this pair of equations, we need to rearrange the first equation a bit. Let's say we want to isolate y on one side and substitute the resulting value into the other equation. To do that, we subtract x from both sides, resulting in y = 14 - x. This equation is still true, as it is a variation of the original equation. We substitute that into the other equation, and we get: 4x - 5(14 - x) = 2. Bingo! Now we only have one variable to deal with, so we can easily solve for x. Let's simplify the equation first. Using the distributive rule, 5(14 - x) = 70 - 5x. But because there was a minus sign before that 5, the equation would be: 4x - 70 + 5x = 2. Simplifying it deeper would make it 9x - 70 = 2. Add 70 to both sides, and the equation becomes 9x = 72. Divide both sides by 9, and we get the value for x, x = 8. Now we'll substitute x = 8 back into the first equation: 8 + y = 14. Subtract 8 from both sides, and boom, we have y = 6. Now we have the only working values of x and y, x = 8 and y = 6. Problem solved! See? Algebra can be fun! Now, let's use the addition/subtraction of equations method, aka elimination method. For this method, what we want to do is multiply one whole equation so that we can add/subtract it with the other equation, in order to eliminate a variable and leave one behind to solve. Scroll up again to see the equations we are using for this example, if you have already forgotten. Let's say we want to eliminate y. The second equation has -5y in it, and to eliminate y, we need to add 5y to get 0. Luckily, we just have to multiply the first equation by 5, so we get: 5x + 5y = 70. Let's add the second equation to this new equation. In this case, y is eliminated, and we are left with 5x + 4x = 70 + 2. Simplify it to 9x = 72, divide both sides by 9, and x = 8 again (obviously). Substitute that into the first equation, and you know the rest. So y = 6 again, duh. Don't trust me? To check whether the x- and y-values are correct, we have to substitute them in both equations and see if they make the equations true. Doing this, we get 8 + 6 = 14, which is correct, and 32 - 30 = 2, which is also correct. We have successfully proven that we have truly solved the problem!
